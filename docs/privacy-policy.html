Privacy Policy for GIF Vault

Last updated: March 14, 2026

GIF Vault is a browser extension that lets users save GIFs and short media into a local browser vault.

Information GIF Vault Handles

GIF Vault may process:

  • Media URLs entered by the user
  • Media and page data needed to import content the user explicitly selects
  • Saved media files and related metadata such as names, favorites, and source references
  • Extension logs and storage-usage information
  • Extension settings such as theme preference and import state
  • Host-permission grants explicitly approved by the user for optional import domains

How Information Is Used

GIF Vault uses this information only to provide the extension's core functionality, including:

  • Importing media from supported pages and URLs chosen by the user
  • Resolving supported Twitter/X media links
  • Checking video duration and converting supported video media to GIF locally when needed
  • Displaying, organizing, searching, renaming, favoriting, copying, and deleting saved items
  • Showing logs and extension storage information
  • Preserving extension settings and import progress

Storage

Saved media and related extension data are stored locally in the user's browser profile using browser storage technologies such as IndexedDB and extension storage.

Data Sharing

GIF Vault does not sell user data.

GIF Vault does not use user data for advertising.

GIF Vault does not use analytics or tracking services.

GIF Vault does not transfer user data to third parties except when necessary to request media or page resources from a website the user explicitly chose to import from.

Remote Services

When a user imports media from a supported website or URL, GIF Vault may contact that site (and related resolver endpoints for supported social-media imports) to retrieve media or related page resources needed to complete the user-initiated import.

GIF Vault does not require user accounts and does not provide account-based services.

User Control

Users can delete saved media, logs, and other local extension data from within the extension where supported, or by uninstalling the extension and clearing extension data from the browser profile.

Changes to This Policy

This privacy policy may be updated to reflect product changes. Any updates will be published at this same policy location.
